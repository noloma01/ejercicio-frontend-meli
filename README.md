# ejercicio-frontend-meli
Prueba técnica Frontend Engineer | Mercado Libre

**Versiones a instalar**

- Node: v10.15.0
- NPM: v6.4.1
- Angular: v7.2.8
- AngularCLI: v7.3.5


> **Desarrollo**

Posicionado en **/ejercicio-frontend-meli:

* *Servidor:* 
1. `cd servidor`
2. `npm run devserver`

* *Cliente:* 
1. `cd cliente`
2. `npm run devclient`


> **Producción**

Posicionado en **/ejercicio-frontend-meli:

1. `cd cliente`
2. `npm run prodclient`
3. `cd ../servidor`
4. `npm run prodserver`
5. Ingresar a http://localhost:3000
